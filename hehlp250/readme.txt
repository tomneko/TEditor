hedit.cnt            ヘルプの目次ファイル

----------------------------------------------------------------------
 ３．実行方法
----------------------------------------------------------------------
lhasa などの LHA 解凍ツールで解凍してください

hedit.hlp の 「Delphi IDE へのヘルプファイルの登録」のトピックには
Delphi-IDE に hedit.hlp をインストールする手順が記述されています。

hedit.hlp を Delphi ヘルプシステムへインストールすると、設計中の
フォーム上や、コードエディタ内、そしてオブジェクトインスペクタ上からも
Ｆ１キーを押すだけで TEditor 関連のヘルプトピックが表示されるようにな
ります。
